# tsp_aed
Resolution of the traveling salesman problem.

## To compile
Use makefile:
```
$ make tsp
```
To compile bruteforce approach:
```
$ make bruteforce
```
## Run
Open a terminal and run:
```
$ ./tsp -f
```
To run bruteforce approach open a terminal and run:
```
$ ./bruteforce
```
## Clean
To clean old builds run:
```
$ make clean
```
